# llm_book_wip
A human assisted AI authored crash course on AI stuff

The project is incarnated as a Claude Projects project, and the folders project_knowledge and project_instructions correspond to their counterparts in Claude Projects.
The creation method itself has been designed by Claude and is reported in book_production_recipe.md.
The syllabus of the course is in the knowledge folder and has been redacted by Claude and GPT5 in a preliminary brainstorming session.
